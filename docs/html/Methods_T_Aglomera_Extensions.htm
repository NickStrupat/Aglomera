EvaluateClusteringTInstance(ClusteringResultTInstance, IInternalEvaluationCriterionTInstance)
Evaluates all given ClusterSetTInstances according to the given IInternalEvaluationCriterionTInstance. The maximum number of clusters allowed in a cluster-set for it to be evaluated corresponds to sqrt(N/2), where N is the total number of instances clustered.

Returns the medoid of a given ClusterTInstance, i.e., a representative object whose dissimilarity to all the instances in the cluster is minimal.
